# 🎓 Student Placement Tracker

A simple web application to track student placement details.

## 🚀 Features
- Add student name
- Add company name
- Add placement status
- Dynamic table update using JavaScript

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript

## 📌 How It Works
Enter student details in the form and click "Add".
The data will be displayed in the table dynamically.
